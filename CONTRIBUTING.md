# Contributing to Channel Fracture

## 学术使用

学术研究可自由使用、修改、复制本仓库代码。欢迎提交 Issue 和 Pull Request 来改进 MVP 示例和文档。

## 商业授权

商业用途（包括但不限于：集成到商业产品、提供商业服务、作为商业项目的一部分）必须获得作者的明确书面授权。

### 联系方式

- 通过 GitHub Issue 提交商业授权请求：https://github.com/ADE-standard/channel-fracture/issues
- 或联系作者 Dexing Liu（刘德星）

## 贡献指南

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/your-feature`)
3. 提交更改 (`git commit -m 'Add your feature'`)
4. 推送到分支 (`git push origin feature/your-feature`)
5. 创建 Pull Request

提交 PR 即表示您同意您的贡献受本仓库 LICENSE（All Rights Reserved）条款约束。
